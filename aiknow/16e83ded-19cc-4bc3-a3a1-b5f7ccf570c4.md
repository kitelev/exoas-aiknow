# Wash-out / re-challenge стратегия — clinical attribution при отмене 2+ препаратов

## Правило

Когда user сообщает: «у меня было `<симптом>`, я отменил `<препарат A>` и `<препарат B>`, симптом ушёл» — **виновник unknown** (один из двух, или комбинация). Не паниковать «оба обратно» и не делать «оба permanent drop». Применить wash-out / re-challenge protocol:

1. **W₀ wash-out (5-7 дней clean):** минимальная стабильная база
	- Drop оба подозреваемых препарата
	- Drop fellow-traveler высокого риска (CYP/serotonergic/sympathetic нагрузка) для чистоты эксперимента
	- Hold безопасные/нейтральные препараты (D3, K2, цетиризин — не trigger'ы)
	- Снизить вспомогательные стимулирующие на ½ дозы (CleverDRIVE → ½, magnesium → ½ если orthostatic подозрение)

2. **W₁ re-challenge — реинтродукция по одному:**
	- D1: вернуть **наименее вероятного виновника** (для control). Наблюдение 3-5 days.
	- D5: если ok → вернуть **следующий по нисходящей вероятности**. Наблюдение 3-5 days.
	- Самого вероятного виновника — последним, или **НЕ возвращать до анализов** (clinical signal trumps protocol-completion).

3. **Усиленный monitoring во время W₀-W₁:**
	- Orthostatic test ежедневно (АД лёжа → стоя; ΔСАД>20 / ΔЧСС>20 = orthostatic hypotension)
	- ЧСС в покое утром
	- Log эпизодов с timestamp + триггер (положение тела, время после еды, какие препараты)

4. **Триггеры для escalation:**
	- Повтор симптома на wash-out схеме → НЕ самолечить, к врачу
	- Synkope, neurological signs (asymmetry, speech) — немедленно

## Why

- **Инцидент 2026-05-16 (Курс БАД W21):** User сообщил vertigo Sun W20 + Mon W21 D1, разрешился после отмены Цитофлавина + Омеги-3. Без wash-out протокола — два возможных fail modes: (a) вернуть оба → vertigo повторится без знания виновника; (b) исключить оба permanent → потеря потенциально полезных компонентов.
- Wash-out даёт **clean attribution** для будущих решений.
- Без минимизации fellow-travelers (зверобой serotonergic, CleverDRIVE йохимбин) — шум в эксперименте.

## How to apply

- При user-report подобной ситуации — **обязательно** предложить wash-out structure, не "вот пересмотренный список".
- Конкретные decisions (что drop, что hold, что cut в ½) — interview-style, не batch (один вопрос за раз):
	1. Сон ok? → flag перегрузки ЦНС
	2. АД/ЧСС? → flag sympathetic/hypotensive risk
	3. Энергия/фокус? → flag CleverDRIVE-class effects
	4. Побочные (ЖКТ/кожа/тревожность)? → flag specific systems
- Disclaimer **обязателен:** "Это not medical advice. При повторе или сомнении — врач первичен."
- Re-challenge план — в body ассета (machine-readable), не в эфире chat'а.
- Triggered также для: новые антидепрессанты, новые сосудистые, новые adaptogens.
