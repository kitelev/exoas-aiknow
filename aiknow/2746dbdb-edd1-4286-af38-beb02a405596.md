## Trigger logic
Frontmatter валидируется только если `looksLikeAsset = "exo__Instance_class" in fm || "exo__Asset_uid" in fm`. Plain markdown notes без этих ключей passthrough.

## Instance_class wikilink shape check
Для namespaced classes (`exo__`, `ems__`, ...): wikilink target не должен содержать whitespace или `()`. Для multi-valued — INVALID если **все** entries malformed (mixed array — bad дропаются, valid сохраняются per Issue #3121).

## isEmpty helper
```typescript
const isEmpty = (v) => {
  if (v === null || v === undefined) return true;
  if (typeof v === "string") return v.trim() === "";
  if (Array.isArray(v)) return v.length === 0 || v.every(isEmpty);
  return false;
};
```
